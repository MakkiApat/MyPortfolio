What was our Project?

This project is a proof-of-concept web application for the Philippine Federation of Married Catholic Priests (PFMCP), designed to replace their manual record-keeping with a centralized digital platform. It aims to streamline membership and organization management, improve communication and data accuracy, and modernize administrative processes. Key features include member and organization registration, event management, admin dashboards, and a robust database with API integration. By providing real-time access to information and enhancing data security, this web application seeks to improve the overall efficiency and accessibility of the PFMCP.

Purpose

The primary purpose of this web application is to:

  • Streamline Membership and Organization Management: Facilitate efficient member registration, organization accreditation, and data management.
  • Improve Communication and Accessibility: Provide real-time access to information for both administrators and members, including event updates and membership details.
  • Enhance Data Security and Accuracy: Implement a secure digital database to reduce errors and improve data integrity.
  • Modernize Administrative Processes: Transition from manual, paper-based records to a digital platform, enhancing overall organizational efficiency.

Key Features

User-Side Features:

  • Member registration form for aspiring members.
  • Organization registration form.
  • Event viewing for members, displaying upcoming and current events.
  • Public facing pages such as About us, and a home page.

Admin-Side Features:

  • Admin dashboard for managing members, organizations, and events.
  • Member and organization approval and tracking.
  • Event creation and management (adding new events).
  • Data retrieval and display from the database.
  • Secure login.

Database Management:

  • A comprehensive database schema with interconnected tables for secure data storage.
  • API for data retrieval and manipulation.

Technology Stack:

  • Front-end: HTML, CSS, JavaScript.
  • Back-end: PHP.
  • Database: MySQL (XAMPP for local, Freedb.tech for remote).
  • IDE: VSCode.
  • Testing: Chrome, Firefox.

API Functionality:

  • API endpoints that retrieve specific records from the database.

Technology and Resources

The proof of concept for the Philippine Federation of Catholic Married Priests website employs a set of standard yet effective tools. VSCode serves as the primary Integrated Development Environment (IDE), chosen for its efficiency in code writing and debugging, thereby streamlining the development process. For local database management and testing, Xampp with MySQL is utilized, enabling thorough evaluation of database functionality before live deployment. The remote database will be hosted on Freedb.tech, ensuring consistent accessibility and data persistence for the deployed application. Standard web browsers, such as Chrome and Firefox, are employed to test the front-end design and user interface, guaranteeing a user-friendly experience. PHP is implemented for server-side scripting and database interaction, facilitating dynamic content generation. HTML, CSS, and JavaScript are used to construct the front-end interface, focusing on both visual appeal and interactive functionality. For the development phase, a standard personal computer or laptop is sufficient. Upon deployment, a web hosting service will be necessary. These tools and resources collectively contribute to the construction of a robust and functional website, facilitating thorough testing and a reliable online presence.
